üì¶ Props

Props are passed as attributes and accessed via $props

Component Definition:

component Card()
  - const { title, variant } = $props
  div(class=`card-${variant}`)
    h3= title

Usage:

Card(title="Hello" variant="primary")

üé∞ Named Slots

Components can have multiple named slots for flexible content insertion

Component with slots:

component Card()
  .card
    slot(content)
    if hasFooter
      .footer
        slot(footer)

Usage with slot content:

Card(hasFooter=true)
  slot(content)
    p Main content
  slot(footer)
    button Click me

Result:

üîç Conditional Slots (if/else)

Slots can be placed inside control structures like if/else statements

Component with conditional slot:

component Card()
  - const { hasFooter = false } = $props
  .card
    slot(content)
    if hasFooter
      .footer
        slot(footer)
